# Firn

A interactive ls.
> Only works on Linux.

## Build

`make all`

## Usage

`firn [path]`
If no path is provided the current directory will be opened.<br>
Deleted files are moved to ~/.trash<br>

## Key Bindings

* h (up directory) 
* j (down)
* k (up)
* l (down directory)
* d (delete)
* SPACE (select)
